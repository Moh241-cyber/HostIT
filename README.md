# HostIT - Plateforme de Cloud Hosting

**Portfolio Mohamed YAYA Ali**  
Projet complet de plateforme cloud pour hébergement, déploiement d'applications, gestion de ressources et scalabilité automatique.

## Fonctionnalités principales
- Gestion dynamique de serveurs (CPU, RAM, stockage, bande passante)
- Déploiement automatique d'applications web (via Docker ou CI/CD)
- Dashboard utilisateur avec authentification (Redux + auth)
- Upload et gestion de fichiers (quota, activity log)
- Intégration cloud (AWS / Azure / Google Cloud, Terraform, Docker)
- Monitoring et scalabilité horizontale

## Architecture
- Frontend : React + Redux (sous-module frontend/hostit-frontend)
- Backend : Node.js / Express (ou ton stack)
- Infrastructure : Docker, Kubernetes / Terraform pour provisioning

## Compétences démontrées
- Architecture des systèmes d'information et cloud computing
- Développement full-stack (front + back)
- Gestion de sécurité (auth, quota)
- DevOps (Dockerfile, déploiement, scalabilité)
- Lien avec mes activités de consulting IA/data/cloud et formations
